# 深度阅读提示词模板

执行三步法时，直接复制使用以下提示词模板，替换 `{书名}` 和 `{章节}` 等占位符。

---

## 一、Step 1 — 结构分析提示词

### 基础版（用户已有目录）

```
这是《{书名}》的目录：

{粘贴目录内容}

请帮我分析：
1. 全书的顶层结构逻辑（核心分为哪几大模块，模块之间是什么关系）
2. 作者的论述展开方式（如：从个人经历到抽象原则，还是从问题到解决方案）
3. 建议的阅读顺序和重点章节
4. 读完整本书后，我应该形成的3-5个核心认知

请用简洁的结构化方式呈现。
```

### 增强版（需要AI搜索书籍信息）

```
请帮我搜索《{书名}》的完整目录或大纲，然后：

1. 分析全书的顶层结构逻辑（核心分为哪几大模块，模块之间是什么关系）
2. 作者的论述展开方式（如：从个人经历到抽象原则，还是从问题到解决方案）
3. 建议的阅读顺序和重点章节
4. 读完整本书后，我应该形成的3-5个核心认知

如果无法找到完整目录，请基于已有信息提供尽可能详尽的结构分析。
```

## 二、Step 2 — 问题生成提示词

### 章节问题生成

```
这是《{书名}》中"{章节名}"的主要内容概述：

{粘贴章节内容或概述}

请基于这部分内容，为我生成5-10个深度阅读问题。要求：
1. 不要简单的事实性问题（如"作者说了什么"）
2. 聚焦"为什么"和"如何应用"（如"作者为什么这样认为？""这个道理如何应用在我的工作中？"）
3. 问题之间要有递进关系，从理解到应用
4. 至少包含1个需要我结合自身经验思考的问题

格式：每个问题标注【理解类】、【应用类】或【反思类】。
```

### 主题问题生成（针对特定概念）

```
我想深入理解《{书名}》中的"{核心概念}"这个概念。

已了解的内容：
{粘贴相关段落或笔记}

请帮我生成一组问题（5-7个），帮助我真正吃透这个概念。要求：
1. 从表层含义到深层逻辑逐层深入
2. 包含与其他概念的关联问题
3. 至少2个应用到实际场景的问题
4. 至少1个挑战或反思性的问题（如"这个概念的边界/反例是什么？"）
```

## 三、Step 3 — 主动回忆提示词

### 标准回忆测试

```
我刚读完《{书名}》的"{章节名}"，这是我的阅读笔记：

{粘贴笔记或重点摘录}

请基于这章的核心内容，生成5个回忆性问题。要求：
1. 覆盖本章最重要的概念、逻辑和结论
2. 问题应该需要"理解"而非"背诵"才能回答
3. 至少1个需要跨段落/跨概念综合的问题

重要：请一次只问1个问题。等我回答后，再出下一题，并给我的回答反馈。
```

### 严格提取练习（强化记忆）

```
请基于《{书名}"{章节名}"的内容，对我进行一次严格的提取练习。

规则：
1. 我会闭上眼睛/合上书，尝试用自己的话复述核心内容
2. 你提出一个引导性问题，我回答后，你帮我检查：
   - 我说对了什么（强化）
   - 我遗漏了什么（补充）
   - 我误解了什么（纠正）
3. 共5个问题，难度递进

现在开始第一个问题。
```

### 跨章节综合回忆（复习阶段）

```
我已经读完了《{书名}》的以下章节：
{已读章节列表}

请生成3-5个跨章节的综合回忆问题，要求：
1. 每个问题需要联系多个章节的内容
2. 考察我对全书整体逻辑的理解
3. 至少1个"如果作者面对XX情况，会如何使用不同章节的原则？"类问题
4. 至少1个"这些原则之间有什么矛盾或张力？"的反思问题

一次一问，我回答后给反馈。
```

## 四、深度理解单个概念（辅助提示词）

### 概念深挖

```
我对《{书名}》中的这句话/概念很感兴趣：

"{引用原文}"

但我感觉"觉得有道理"不等于"真正理解"。请帮我：
1. 用大白话解释这句话的真正含义（不是翻译，而是解释底层逻辑）
2. 举一个贴近生活的具体例子说明
3. 举一个职场/专业场景中的应用案例
4. 这句话的反面或边界是什么？（什么时候不适用）
5. 如果要我把这个道理讲给一个没读过这本书的人听，我应该怎么说？

目标是让我能用自己的话把这个概念讲清楚。
```

### 举一反三

```
基于《{书名}》中"{概念}"这个核心思想：

1. 这个思想在其他领域有什么类似的应用？（跨学科迁移）
2. 历史上或现实中，有哪些成功/失败的案例体现了这个原理？
3. 如果我想要实践这个思想，接下来一周我可以做的一件具体小事是什么？
```

## 五、阅读计划制定

### 分阶段阅读计划

```
我想读《{书名}》，根据它的结构和我的时间，请帮我制定阅读计划。

书籍结构：
{粘贴Step 1的结构分析}

我的情况：
- 每天可投入时间：{X}分钟
- 期望完成时间：{X}周内
- 阅读目的：{学习知识/解决实际问题/拓宽认知等}

请输出：
1. 分阶段的阅读安排（每阶段覆盖哪些章节）
2. 每个阶段对应的三步法执行要点
3. 每阶段结束后的自我检验标准
```

## 记忆口诀设计指南

为每个核心概念设计记忆口诀时，遵循以下原则：

1. **用具体类比**：将抽象概念比作日常事物（如"系统就像自动驾驶仪，目标只是导航目的地"）
2. **押韵或节奏**：朗朗上口的短句更容易记住（如"先观察，不评判；讲感受，说需要"）
3. **画面感**：创造 vivid 的视觉画面（如"1%复利就像烧水，99度前看不到变化，但坚持到100度就沸腾"）
4. **情感锚点**：与个人情感经验关联（如"痛苦+反思就像炼金术，把废铁变成黄金"）
5. **简短**：不超过20个字，方便随时回想

## 使用说明

1. **按需调用**：根据当前所处步骤，复制对应提示词，替换占位符后使用
2. **灵活调整**：如果书籍类型特殊（如小说、教材、论文），可适当调整问题角度
3. **迭代优化**：根据AI的初始输出，追加指令细化和深化
4. **强制记忆口诀**：每次生成记忆卡片时，必须为每张卡片设计一个记忆口诀
