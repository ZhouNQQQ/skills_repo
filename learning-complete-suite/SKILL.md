# 学习完整体系 (Learning Complete Suite)

整合费曼学习法、科学学习底层逻辑、学习动力系统三大方法论，
提供从计划设计到深度理解、科学记忆、动力养成的完整学习系统。

## 2. 七步学习计划设计工作流

### Step 1: 需求分析
识别学习目标、评估现状、明确约束条件。输出：学习画像。

### Step 2: 内容拆解
构建知识地图、分析依赖关系、排序优先级。输出：学习路径。

### Step 3: 策略选择

| 场景 | 核心方法 | 辅助方法 |
|------|---------|---------|
| 深度理解 | 费曼学习法 | 间隔重复 + 主动回忆 |
| 长期记忆 | 间隔重复 + 主动回忆 | 睡眠优化 + 测试驱动 |
| 持续执行 | 动力系统 + 习惯养成 | 番茄工作法 + 学习闭环 |
| 考前冲刺 | 测试效应 + 分散练习 | 检索式练习 + 睡眠优化 |
| 技能习得 | 刻意练习 + PBL | 成长型思维 + 同伴监督 |
| 体系构建 | 类比法 + 知识连接 | 思维导图 + 输出式学习 |

### Step 4: 深度理解 — 费曼学习法
以教代学，通过三次复述实现知识内化。详见 references/feynman-workflow.md。

### Step 5: 科学记忆 — 底层逻辑与战术
基于认知科学的六大记忆优化战术。详见 references/science-tactics.md。

### Step 6: 动力与习惯 — 长期执行保障
动力系统（动机+能力+毅力）与六大习惯模块。详见 references/learning-power-habits.md。

### Step 7: 评估迭代
评估阶段目标达成度，根据反馈调整后续策略。

## 3. 约束与边界

1. 不做学科内容教学，只提供学习方法
2. 不替代专业教育，适用于自学和辅助学习
3. 需用户配合执行，方法有效的前提是用户主动行动
4. 效果因人而异，需根据用户反馈动态调整
