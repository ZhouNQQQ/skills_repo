| 字段 | 内容 |

## 完整小传（雪花步骤5）

每个重要角色的"从出生到故事结束"的完整经历。

### 主角完整小传

```
## 出生与童年
（家庭背景、童年创伤、 formative events）

## 成长经历
（关键人生节点、技能习得、人际关系形成）

## 故事前状态
（故事开始时的处境、为什么是现在、触发事件的土壤）

## 故事中的成长弧线
（起点状态 → 触发事件序列 → 渐进改变节点 → 终点状态）

## 故事后的未来
（角色在故事结束后的去向、 legacy）
```

### 小传与人物卡的关系

- **人物卡** = 故事的"简历"（关键信息，一张纸）
- **完整小传** = 故事的"传记"（完整经历，多段）
- 人物卡用于快速参考，小传用于深度理解和一致性保障

## 角色过滤器提问法

写任何角色的行动或对话前，依次提问：
1. 以这个角色的性格，他会这样做/这样说吗？
2. 如果答案是否定，是否有充分的剧情铺垫？
3. 这个行为会导致人际关系/能力边界/记忆承诺发生哪些变化？
4. 这些变化是否需要在后续情节中体现？

## 一致性自检清单

产出角色相关内容后，逐一确认：

- [ ] **人设一致**：行为/语言/决策是否符合核心性格档案？
- [ ] **能力合规**：能力是否在已设定边界内？提升是否有迹可循？
- [ ] **情感合理**：情绪反应是否与触发事件分量匹配？
- [ ] **关系连贯**：互动是否与当前关系层级一致？变化是否有事件支撑？
- [ ] **弱点在场**：弱点/缺陷是否有所体现？
- [ ] **记忆延续**：是否回应了之前的重要承诺/人情/恩怨？
- [ ] **背景在场**：行为是否体现了出身/阶层/文化背景影响？
- [ ] **成长有据**：变化是否由已铺垫的事件推动？程度是否合理？
