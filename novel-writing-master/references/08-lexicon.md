# 词汇修辞技法（写作辞林）

基于袁明光、杜奋嘉、彭会资《写作辞林》的古典写作方法论。

## 执行流程

1. **诊断写作需求**：判断"选词求助"（已有内容需优化）还是"从零描写"（需要生成新文本）
2. **确定描写维度**：从写人/写景/写物/写事四大类中锁定具体子类
3. **选择技法组合**：根据文体和场景选择2-3种核心技法
4. **执行并输出**：按质量标准输出优化后的文本，附简要用词说明

## 四大分类核心原则

| 大类 | 核心原则 | 关键提醒 |
|------|---------|---------|
| 写人 | 以外显内，动态呈现 | 避免直接贴标签，用动作、神态、细节折射性格 |
| 写景 | 情融于景，移步换形 | 景物是情绪的同构物，随视角移动分层展开 |
| 写物 | 形神兼备，赋予生命 | 器物要有"用"的场景，动植物要有动态和习性 |
| 写事 | 张弛有度，细节放大 | 高潮处减速，用特写镜头替代流水账叙述 |

## 核心技法

### 1. 精词优选法

**操作流程**：
- 提取核心意象 → 发散5-8个候选词（分力度层级和情感色彩） → 按文体筛选 → 验证独特性

**禁区**：不得用"非常""很""特别"等程度副词+普通形容词的组合；同一段落核心词不重复超过两次。

### 2. 范例仿写法

- **选范**：根据需求选取对应范例类型
- **拆骨**：分解为词汇层（核心动词/形容词/比喻词）、结构层（描写顺序）、节奏层（句式长短变化）
- **换血**：保留结构和节奏，替换为自己的素材
- **脱形**：修改50%以上句式结构和词汇，融入个人风格

**深度等级**：初级模仿顺序和角度 → 中级模仿词汇搭配和修辞密度 → 高级模仿神韵和氛围，用语完全自创。

### 3. 五感描写法

**应用指南**：
- 视觉主导场景：叠加听觉或触觉提供纵深
- 听觉主导场景：叠加嗅觉或触觉提供空间定位
- 静态场景：用时间性感官（嗅觉"陈年"、触觉"温润"）赋予动感
- 情绪场景：用生理化感官词汇（"胃部收缩""铁锈味"）将抽象情绪具象化

**五感词汇速查**：
- 视觉：晶亮、晦暗、斑驳、绯红、惨白、墨绿、摇曳、迸射
- 听觉：喑哑、嘹亮、细碎、清脆、浑厚、嘶哑、回荡、萦绕
- 嗅觉：馥郁、清冽、若有若无、陈腐、馨香、腥甜
- 触觉：溽热、凛冽、温凉、粗粝、滑腻、绵软、刺痛、酥麻
- 味觉：常用于通感修辞（"苦涩的风""甜腻的阳光"）

### 4. 修辞配合法

| 组合 | 要点 |
|------|------|
| 比喻 + 质感词 | 用触觉/视觉质感词构建喻体（"月光像一块凉玉"），避免"像""仿佛" |
| 拟人 + 动作词 | 选择带主观意图的动词（"风在屋顶徘徊"），忌用"好像在说" |
| 排比 + 节奏词 | 分句核心词音节数保持一致或递进，形成音韵推进感 |
| 对偶 + 反义词 | 利用同一语义场的反义或近义词汇形成张力 |
| 通感 + 跨类词 | 将某一感官词汇移植到另一感官语境（"颜色很响亮"） |

**修辞密度控制**：
- 描写性段落：每100字不超过3处修辞，类型错开
- 抒情性段落：密度可提高，需有"散句"穿插作为呼吸空间
- 叙述性段落：以白描为主，修辞仅用于关键节点

## 输出质量标准

1. **词汇精度**：每个核心意象有至少一个精准刻画的词
2. **画面可感**：读者能在脑海中生成清晰或富有氛围的画面
3. **节奏变化**：长句与短句交错，避免单调感
4. **感官层次**：至少调动两种以上感官（视觉除外）
5. **风格一致**：同一段落内的词汇色彩保持一致，不混搭
6. **无废词**：删除后不影响意义的词汇必须删除
