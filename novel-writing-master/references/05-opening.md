# 开篇设计与钩子

## 六条核心原则

1. **黄金三章法则**：第1章抓人眼球，第2章建立共情，第3章锁定追读
2. **即时冲突**：开篇500字内必须出现冲突，拒绝平铺直叙
3. **主角优先**：主角必须在第1章出场，读者只为主角而来
4. **悬念植入**：首章至少埋下1个短期悬念和1个长期伏笔
5. **期待感建立**：让读者明确"主角要做什么、做到有什么好处"
6. **节奏紧凑**：单段控制在3行以内，每段必须有明确叙事功能

## 六种开篇钩子模板

1. **危机前置型**：主角一上来就面临生死危机
2. **倒叙高光型**：将后期名场面前置，先震撼再回溯
3. **悬念预埋型**：抛出神秘事件/物品/规则，引发追问
4. **反差人设型**：展示主角与众不同的核心特点
5. **世界异变型**：世界规则突然改变，主角被迫适应
6. **强冲突型**：主角与强敌/环境的直接对抗

## 多重危机叠加法

- 极寒 + 风暴 + 丧尸 + 吞噬星渊 + 邪神巨尸 + 永夜
- 多层不同类型危机叠加，快速建立极端压迫感
- 配合明确行动号召（"向东方，逃出黎明"），形成强烈使命感

**执行要点**：
- 危机类型差异化（自然/生物/超自然/社会各1-2种）
- 必须为主角提供明确的生存目标和方向
- 控制叠加节奏，每层之间留有主角反应空间
- 压迫感越强，破局后的爽感越足

## 双Hook章节结构

### 章首Hook（吸引读本章）
- 第一句即抛出冲突、异常或危机
- 拒绝环境描写、背景介绍作为开头
- 直接切入动作或关键对话

### 章末Hook（吸引读下一章）
- 结尾停在危机升级、悬念揭露或重大选择前
- 使用预叙技巧：预告即将发生的重大事件
- 结尾最后一句必须是疑问、危机或反转

## 开篇自检清单

- [ ] 主角是否在第1章前300字出场？
- [ ] 首章是否有明确的冲突/危机？
- [ ] 是否埋设了至少1个需要继续阅读才能解答的悬念？
- [ ] 读者是否清楚主角当前的目标？
- [ ] 章节末尾是否有强Hook促使翻页？
- [ ] 是否存在超过5行的长段落？
- [ ] 是否有"介绍性"文字（背景、设定、世界观解释）挤占了开篇节奏？
- [ ] 人设是否通过行动/对话落地，而非直接定性描述？

## 使用方法

1. 根据题材从六种钩子模板中选择最适合的开篇类型
2. 套用双Hook结构规划每章的首尾节奏
3. 对首章使用自检清单逐项核查
4. 如写末世/灾变题材，可叠加多重危机法强化压迫感
