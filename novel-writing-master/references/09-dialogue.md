# 对话写作：炫人耳目的对话

基于詹姆斯·斯科特·贝尔《如何创作炫人耳目的对话》。

## 核心理念：对话 = 议程 + 冲突

对话的本质是**作为行动的言说**。人物讲话不是为了告诉读者信息，而是为了推进自己的目标。

- **议程（Agenda）**：每个人物在对话中都想要某样东西。没有议程的对话就是闲聊。
- **冲突（Conflict）**：人物之间的议程相互对抗，产生张力。

## 对话的五种功能

1. **呈现故事信息**：先行动后解释，将信息藏入冲突之中
2. **展示人物**：每个人物必须有独特的**声口（Voice）**
3. **奠定基调**：通过语气、用词、节奏确立场景氛围
4. **设计场景**：用对话驱动场景的行动、转折和收束
5. **传达主题**：让主题从对话中自然浮现，不要直接说出

## 五套训练法

| 训练 | 操作 |
|------|------|
| 声口日记 | 以目标人物口吻撰写日记，探索内心声音和词汇习惯 |
| 出声朗读 | 将对话读出声，用听觉检验是否自然、有节奏感 |
| 改写电影脚本 | 选取优秀剧本对话，改写成叙述形式，分析如何让对话更生动 |
| 即席表演 | 选择人物类型，用其口吻说话一天，训练本能表达 |
| 单纯练习 | 设定两个有冲突议程的人物，大量书写纯对话，删除叙述和标签 |

## 四种加强冲突与张力的方法

1. **记住那些议程**：每一句前问自己——这个人想要什么？这句话在推进议程吗？
2. **深入更多的争辩**：让对话者争论起来。友好的一致意见杀死张力。
3. **搭设障碍**：设置物理或心理阻碍，使直接沟通变得困难。
4. **忧惧因素**：引入担忧和恐惧，让人物隐藏真实情绪或暴露脆弱。

## 十一个技术秘方

| 编号 | 秘方 | 核心要点 |
|------|------|----------|
| 1 | 编排 | 给每个人物一两句独特描述和一种怪癖 |
| 2 | 倒转显而易见 | 让对话出乎意料，打破读者预期 |
| 3 | 潜文本 | 让人物语焉不详，话里有话，说东指西 |
| 4 | 廉价香槟法 | 表面的轻松愉快掩盖底下的张力 |
| 5 | 家长/成人/孩子 | 运用交互分析理论设计对话中的权力动态 |
| 6 | 屈弯语言 | 打破常规语法和语序，制造独特节奏 |
| 7 | 从鼻尖上下来 | 避免一切尽在"鼻尖上"的直白对话 |
| 8 | 压缩 | 删除冗余，让每句对话都有力量 |
| 9 | 别忘了沉默 | 用沉默和停顿制造张力 |
| 10 | 控制步调 | 通过问答节奏控制紧张感——急促或延宕 |
| 11 | 宝石与香料 | 偶尔插入金句和独特表达，点睛之笔 |

## 输出质量标准：对话五特征

1. **独特性**：每个人物的声口可辨识，词汇、句式、节奏各不相同
2. **推进力**：每句对话都推进议程或冲突，没有闲聊、没有信息倾倒
3. **潜文本**：人物说的不完全是想的，读者能感受到话外之音
4. **节奏感**：长短句交替，有音乐性，停顿和沉默被有意设计
5. **压缩感**：删除冗余词汇和标签，每句对话承担双重或多重功能

## 执行流程

1. **诊断**：识别当前对话的问题类型（平淡？直白？冗长？缺乏冲突？）
2. **定位**：确定对话中每个人物的 agenda 是什么
3. **设计冲突**：让 agendas 相互对抗
4. **应用技术**：从十一个秘方中选择适用的 2-3 个进行强化
5. **压缩精炼**：删除冗余，检验每句对话是否承载足够功能
6. **质量检验**：对照五个特征标准逐条审查

## 十大议题处理原则

1. **标点**：服务于节奏，不要打断阅读流
2. **引语源与标签**：优先使用"说"，用动作标签替代副词修饰
3. **方言**：少用音标式拼写，多用语法结构和独特词汇暗示
4. **背景故事**：藏入冲突中呈现，不要直接倾倒
5. **内部对话**：与外部对话形成对照，揭示潜文本
6. **实验**：允许打破规则，前提是效果值得
7. **咒骂**：符合人物身份，避免无意义的脏话堆砌
8. **主题式对话**：永远不要让人物直接说出主题
9. **喜剧调剂**：喜剧对话同样需要议程和冲突
10. **旧时期对话**：抓住时代语感，避免现代词汇穿帮
