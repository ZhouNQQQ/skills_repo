# 雪花大纲（故事骨架到详细大纲）

本文件存放雪花写作法从**一句话故事核**到**详细大纲**的全部产物。是知识库中最核心的"骨架文件"。

---

## 步骤1：一句话故事核（15字以内）

```
（谁）+（要做什么）+（代价/阻碍）
```

当前版本：
```
________________________________________________________________________
（待填写，15字以内）
```

**冻结规则**：故事核一旦确定，全书写作不得偏离此核心。如必须修改，在 decisions/ 记录原因。

---

## 步骤2：五句段概括（全书骨架）

```
第一句（背景）：________________________________________________________________________
第二句（灾难1）：________________________________________________________________________
第三句（灾难2）：________________________________________________________________________
第四句（灾难3）：________________________________________________________________________
第五句（结局）：________________________________________________________________________
```

**每句对应一幕**：
- 第一句 → 第一幕铺垫（约20%）
- 第二、三句 → 第二幕对抗（约60%）
- 第四句 → 第二幕高潮转折
- 第五句 → 第三幕结局（约20%）

## 步骤6：详细大纲（场景清单）

每幕扩写成带**场景清单**的详细剧情。一个场景 = 一个地点 + 一个时间点 + 一个目标 + 一个冲突。

|----------|------|------|--------|----------|----------|----------|------|----------|----------|

**场景清单字段说明**：
- **场景目标**：角色在这个场景里想要什么（议程）
- **冲突/阻碍**：什么阻止角色达成目标
- **出场角色**：该场景所有出场人物
- **情绪**：场景的整体情绪走向（+爽 / 0平 / -虐 / ++高潮）
- **揭示设定**：本场景首次向读者揭示的设定（同步到 worldbuilding.md）
- **埋设伏笔**：本场景埋下的伏笔编号（同步到 foreshadowing-log.md）

## 卷级大纲（网文多卷扩展）

网文通常超过100万字，需要多卷结构。每卷独立执行一次雪花步骤6（详细大纲+场景清单）。

| 卷号 | 卷名 | 起止章节 | 雪花步骤 | 场景数 | 核心事件 | 状态 |
|------|------|----------|----------|--------|----------|------|
| 1 | | 1-50 | 已完成 | | | 已完成 |
| 2 | | 51-100 | 待完成 | | | 未开始 |

每卷的详细大纲格式同上，放在本文件下方。

## 雪花步骤完成追踪

```
- [ ] 步骤1：一句话故事核
- [ ] 步骤2：五句段概括
- [ ] 步骤3：雪花人物卡（全部重要角色）
- [ ] 步骤4：扩情节（五幕结构）
- [ ] 步骤5：扩人物（完整小传）
- [ ] 步骤6：详细大纲+场景清单（第一卷）
- [ ] 步骤7：写正文（第一卷）
```

**网文适配规则**：
- 步骤1-3在**动笔前**必须完成
- 步骤4-5可与**连载同步**（提前准备1-2卷）
- 步骤6每卷**开始前**做一次
- 步骤7每章对照场景清单写作，同时保持**30%弹性空间**
