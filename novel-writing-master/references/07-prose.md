# 文笔精细化

## 六条核心原则

1. **展示优先**：用感官细节让读者自己感受，不替读者下结论
2. **白描为核心**：克制形容词，以动作、神态、环境细节推进叙事
3. **去主观化**：减少"他看到""他感到"等视角标签，让读者直接进入场景
4. **去冗余**：副词、模糊状态词、无意义连词尽量精简
5. **五感沉浸**：优先用视觉、听觉、触觉、嗅觉、味觉的细节营造画面感
6. **情绪克制**：收大于放，强情绪优先通过动作和微表情流露，少直白抒情

## 七大精细化技法

### 1. 白描五感叙事法
用五感细节替代概括性形容。优先展示，少说定论。
- 差：他很害怕。
- 优：他指节发白，喉咙发紧，呼吸声在死寂里格外重。

### 2. 副词精准替换与去冗余
剔除弱化张力的副词，用精准动词或细节替代。
- 差：他突然猛地冲了出去。
- 优：他撞开椅子，冲了出去。

### 3. 模糊状态词清零
剔除"感觉""显得""好像""有点"等模糊词，用动作和状态呈现。
- 差：他似乎有点紧张。
- 优：他反复摩挲袖口，目光躲闪。

### 4. 视角去主观化
减少"看到""听到""想到"等主观视角词，叙事直接推进。
- 差：他看到门缝里透进一道光。
- 优：门缝里透进一道光。

### 5. 助词连词极简
精简"然后""就""的/地/着/了"，保留必要通顺即可。非禁用，是节制。
- 差：然后他就开始慢慢地走了过去。
- 优：他走了过去。

### 6. 中文词汇精准适配
优先用中文语境下的精准动词和意象词，避免西化表达和抽象大词。
- 差：他进行了迅速的移动。
- 优：他矮身疾掠。

### 7. 情绪克制表达（收大于放）
强情绪场景优先用动作+微表情+极简台词，克制直白抒情。
- 差：他悲痛欲绝，泪如雨下。
- 优：他背过身去，肩膀抖了一下，没有声音。

## 对话打磨三原则

1. **前缀极简**：对话前缀只保留必要的情绪/动作提示，"他说"够用时不加修饰
2. **人设通过对话立住**：台词本身承载人物身份、性格、处境
3. **功能优先**：每句对话推动剧情、揭示人设或埋伏笔

## 排版与段落规范

- **单段单核心**：每段承载一个功能（一个动作、一句台词、一个画面、一个心理闪念）
- **段落功能明确**：推进剧情、塑造人物、埋设伏笔、渲染氛围——至少占一样
- **合理分段**：网文以手机阅读为主，段落宜短不宜长。单段1-3行，氛围渲染可弹性放宽

## 常见文笔问题速查

| 问题 | 修正方向 |
|------|---------|
| 形容词堆砌 | 用白描细节替代 |
| 副词泛滥（突然、渐渐、开始） | 删或换精准动词 |
| 模糊状态词（感觉、好像、有点） | 用动作/神态/环境呈现状态 |
| 视角标签过多（他看到、他感到） | 删除标签，直接进入场景 |
| 心理描写冗长 | 压缩到核心闪念，或用动作暗示 |
| 台词前缀臃肿 | 极简前缀，靠台词立人设 |
| 助词冗余（然后、就、着、了） | 精简，保通顺即可 |
| 西化/抽象大词 | 换中文语境精准动词和意象 |

## 弹性原则

氛围渲染、抒情、意识流等特殊场景可适度放宽部分规则，功能永远优先于形式。
