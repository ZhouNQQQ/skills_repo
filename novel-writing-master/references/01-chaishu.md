# 拆书学习

系统化拆书学习方法论。拆书不是"看书"，而是**逆向工程+刻意练习+认知建构**。

## 核心原则

1. **拆书≠记情节** —— 重点在拆框架、情绪与爽点的表现技巧
2. **边拆边写** —— 拆完1章仿写1个片段，内化是唯一路径
3. **阶段适配** —— 不同阶段用不同方法，不同平台有不同侧重
4. **拆神不拆形** —— 学习技巧逻辑，而非复制具体剧情

## 四步上手流程

1. 选一本近3个月的同题材爆款（目标平台榜单Top10）
2. 读第1章，逐格填写拆解表
3. 标注情绪值和爽点（只拆这两个维度，其他忽略）
4. 读完3章后，总结"最心动的3个技巧"，立即仿写1个段落

## 极简拆书模板

| 章节 | 场景 | 核心事件 | 情绪值变化 | 爽点类型 | 伏笔/钩子 | 可学技巧 |
|------|------|----------|-----------|----------|-----------|----------|
| 1 | | | | | | |
| 2 | | | | | | |
| 3 | | | | | | |

**情绪值**：用 `0→-→++` 格式，如"0（平静）→-（危机）→+（金手指）→++（反杀）"

**爽点类型**：显摆/打脸/获得/英雄救美/震惊/逆袭/迪化/其他

## 拆书方法选择

| 阶段 | 方法 | 侧重 | 耗时 |
|------|------|------|------|
| 新手 | 八步拆书法 | 单章/开篇结构 | 2-4h/本 |
| 成长期 | 紫丿川三点论 | 框架+情绪+爽点 | 1-2h/本 |
| 签约冲刺 | 黄金三章拆书法 | 开篇钩子+节奏 | 30min/本 |
| 系统提升 | RIA便签拆书法 | 技巧→仿写→应用 | 持续 |

## 四步转化闭环

```
拆解 → 提炼 → 仿写 → 复盘
```

- 段落仿写：拆完1章，仿写精彩段落
- 章节仿写：拆完3-5章，按模板写对应章节
- 结构仿写：拆完前50章，搭建全书框架独立创作
- 复写对比：不看原文重写，逐句对比找差距

## 常见误区

| 误区 | 正确做法 |
|------|----------|
| 拆书=记录剧情梗概 | 只记录框架、情绪与爽点技巧 |
| 拆大神的书学写法 | 新手拆近期爆款（套路裸露），进阶拆大神 |
| 只拆不写 | 边拆边写，拆1章仿1段 |
| 追求拆书数量 | 一本拆透胜过十本拆完 |
| AI一键拆完就完事 | AI拆骨架+人工填血肉 |
| 拆太多反而不会写 | 每次只拆一个维度，拆完立即应用 |

## 输出标准

- 必须包含：**章节结构表** + **情绪曲线** + **爽点分布**
- 进阶增加：人设档案 + 伏笔追踪 + 文笔亮点
- 仿写产出须与原文字数相当，替换为原创场景
- 总结须提炼出**可复用的创作公式**（如：打脸=觉得不行→其实行→反应）
