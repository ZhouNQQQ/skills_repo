# 五维检查法详细清单

## D1 — 深度维度 (Depth)：本质穿透检查

**核心来源**：第一性原理 + 毛泽东思想「透过现象看本质」

**哲学基础**：
- 第一性原理：剥离表象和传统经验，将问题解构到最基本、不可再分的真理，然后从零重构
- 毛泽东「十六字诀」：去粗取精、去伪存真、由此及彼、由表及里

### 检查清单

- [ ] **原点思考**：文章是否从问题的最基本事实出发进行推理？还是依赖类比、比喻、行业惯例？
- [ ] **表象 vs 本质**：文章是停留在现象描述（"发生了什么"），还是揭示了内在规律和根本机制（"为什么会发生"）？
- [ ] **去粗取精**：文章是否从大量信息中筛选出真正关键的材料？还是堆砌无关信息制造信息冗余？
- [ ] **去伪存真**：文章是否辨别了假象与真象？还是被表面现象迷惑？
- [ ] **由此及彼**：文章是否孤立地看待问题？还是联系了相关的背景、前因后果、系统互动？
- [ ] **由表及里**：文章是否深入挖掘了表层之下的深层结构和因果关系？还是浅尝辄止？
- [ ] **类比依赖度**：文章是否大量使用类比推理（"就像……""好比……"）作为核心论证？类比推理只能产生微小迭代，不能触及本质

### 快速判断信号

| 低价值信号（D1 Fail） | 高价值信号（D1 Pass） |
|----------------------|----------------------|
| 大量类比和比喻，缺少本质分析 | 从基本原理出发推导 |
| 就事论事，不追问"为什么" | 连续追问至根本原因 |
| 堆砌现象，不做筛选加工 | 经过"十六字诀"式加工 |
| 复述常识和行业共识 | 挑战共识，揭示隐藏结构 |
| 标题党，情绪渲染 | 冷静分析，层层递进 |

## D2 — 事实维度 (Data)：实事求是检查

**核心来源**：毛泽东思想「实事求是」「调查研究」+ 批判性思维「证据评估」

**哲学基础**：
- 「实事」就是客观存在的一切事物，「是」就是客观事物的内部联系，「求」就是我们去研究
- 没有调查，没有发言权
- 一切结论产生于调查情况的末尾，而不是在它的先头

### 检查清单

#### 材料来源审查
- [ ] **一手 vs 二手**：文章使用的是第一手材料（原始数据、原始访谈、亲自调查）还是转手多次的信息？
- [ ] **来源透明度**：材料来源是否清晰标注？能否追溯到原始出处？
- [ ] **利益相关检查**：信息来源是否与文章结论有利益关系？（如品牌方自己做的功效实验）
- [ ] **专家相关性**：如果引用专家，该专家是否在其专业领域内发言？

#### 数据质量审查
- [ ] **数据来历**：数字是否有明确来源？还是"研究显示""据统计"等模糊表述？
- [ ] **平均值类型**：是平均数、中位数还是众数？三者差异可能极大
- [ ] **全距与分布**：是否只给了平均值而隐藏了分布范围？
- [ ] **基数问题**：绝对增长数字是否考虑了基数大小？（"增长1000人"但基数是100万）
- [ ] **比较完整性**：是否有意省略了对比数据？
- [ ] **时间跨度**：数据的时间范围是否足够支撑结论？

#### 调查研究审查
- [ ] **调查基础**：文章是否经过了真正的调查研究？还是仅凭想象、传闻、网络碎片？
- [ ] **结论顺序**：结论是在调查之后产生，还是先下结论再找证据？
- [ ] **典型性**：个案是否能代表整体？是否"解剖麻雀"又有全局视野？
- [ ] **反面材料**：是否主动寻找了可能推翻自己结论的材料？

### 快速判断信号

| 低价值信号（D2 Fail） | 高价值信号（D2 Pass） |
|----------------------|----------------------|
| 来源不明或来源有偏 | 来源清晰，可追溯 |
| 只有个人经历/典型案例 | 有系统性数据或研究支撑 |
| 数据无来源或来源利益相关 | 数据来自独立第三方 |
| 先下结论后找证据 | 证据先于结论 |
| 选择性引用，隐瞒反面 | 正反两面材料都呈现 |

## D3 — 逻辑维度 (Logic)：论证结构检查

**核心来源**：批判性思维「论证分析/谬误识别」+ 《学会提问》「十大关键问题」

**哲学基础**：
- 论证 = 理由 + 结论
- 没有支撑理由的结论只是"断言"，不是"论证"

### 检查清单

#### 结构完整性
- [ ] **结论清晰**：30秒内能否用一句话概括"作者想让我相信什么"？
- [ ] **理由存在**：每个结论后面是否有"因为……"支撑？
- [ ] **理由相关**：理由与结论之间是否存在逻辑关联？
- [ ] **推理步骤**：从理由到结论的每一步是否都成立？有无逻辑跳跃？

#### 概念清晰度
- [ ] **关键词定义**：核心概念（如"中产阶级""健康发展""优化"）是否有明确定义？
- [ ] **歧义检查**：如果关键词换一种理解方式，结论还成立吗？
- [ ] **概念一致性**：同一概念在文中是否始终使用同一含义？

#### 谬误扫描（详见 fallacy_quick_ref.md）
快速扫描以下高信号谬误：
- [ ] 人身攻击 / 基因谬误（攻击人而非观点）
- [ ] 稻草人（歪曲对方观点再攻击）
- [ ] 虚假两难（"要么A要么B"忽略CDEF）
- [ ] 滑坡谬误（"一旦A就会一路滑到Z"）
- [ ] 诉诸公众（"大家都这么认为"）
- [ ] 诉诸情感（用情绪替代理由）
- [ ] 诉诸可疑权威（跨界引用）
- [ ] 事后归因（"A在B前，所以A导致B"）
- [ ] 因果混淆（相关≠因果）
- [ ] 循环论证（结论当理由）

#### 论证模型检查（图尔敏六要素）
- [ ] **根据(Data)**：支撑主张的事实是否真实可靠？
- [ ] **担保(Warrant)**：从根据到主张的推理桥梁是否成立？
- [ ] **支撑(Backing)**：担保背后的依据是否坚实？
- [ ] **反驳(Rebuttal)**：是否考虑了例外和反对意见？
- [ ] **限定(Qualifier)**：结论是否过于绝对？是否承认适用边界？

### 快速判断信号

| 低价值信号（D3 Fail） | 高价值信号（D3 Pass） |
|----------------------|----------------------|
| 结论模糊或找不到 | 结论清晰，一句话可概括 |
| 无理由支撑 | 有明确理由和证据链 |
| 理由与结论无关 | 推理链条严密 |
| 存在明显逻辑谬误 | 逻辑严谨，无谬误 |
| 完全无视反面意见 | 主动回应反面观点 |
| 绝对化结论 | 结论有条件限制 |

## D4 — 假设维度 (Assumption)：隐含假设检查

**核心来源**：第一性原理「质疑所有假设」+ 《学会提问》「识别价值观假设和描述性假设」

**哲学基础**：
- 第一性原理的第一步就是识别并质疑所有假设
- 任何论证都有隐含假设；如果假设不成立，论证立即崩溃

### 检查清单

#### 假设识别
- [ ] **逻辑鸿沟检查**：从理由到结论之间，需要相信什么额外信念才能让推理成立？
- [ ] **价值观假设**：作者是否隐含了某种价值偏好（如"效率比公平更重要"）？
- [ ] **描述性假设**：作者是否隐含了某种关于世界是什么样子的信念？
- [ ] **行业共识检查**：文章是否将行业常识当作不可质疑的真理？
- [ ] **时间假设**：文章是否假设了当前的规则/条件会永远不变？

#### 假设挑战
- [ ] **否定测试**：如果核心假设被否定，结论是否仍然成立？
- [ ] **替代假设**：是否存在其他同样合理的假设可以支撑不同的结论？
- [ ] **历史检验**：这个假设在历史上是否被验证过？
- [ ] **物理定律检验**：这个假设是基于物理/数学真理，还是基于惯例和约定？

#### 第一性原理追问
- [ ] **五个为什么**：对文章核心观点连续追问"为什么"，看能否追溯到不可再分的基本事实
- [ ] **白痴指数**：文章中的结论与基本事实之间的差距有多大？（越大越可疑）
- [ ] **类比剥离**：如果剥去所有类比和比喻，论证还剩下什么？

### 快速判断信号

| 低价值信号（D4 Fail） | 高价值信号（D4 Pass） |
|----------------------|----------------------|
| 假设完全未被识别 | 明确暴露核心假设 |
| 将行业共识当作真理 | 挑战行业基本假设 |
| 理由→结论跳跃无说明 | 填补了逻辑鸿沟 |
| 建立在脆弱假设上 | 假设经过检验和辩护 |
| "从来如此，便对吗？" | 追溯至不可再分事实 |

## D5 — 实用维度 (Utility)：实践价值检查

**核心来源**：毛泽东思想「实践论」+ 《学会提问》「省略信息/多元结论」

**哲学基础**：
- 实践是检验真理的唯一标准
- 理论必须联系实际，否则就是空洞的教条
- 有什么重要信息被省略了？能得出哪些合理的结论？

### 检查清单

#### 可检验性
- [ ] **实践检验**：文章观点是否可以通过实践来验证或证伪？
- [ ] **可证伪性**：是否存在什么证据可以证明文章是错误的？（如果怎么说都有理，则不可信）
- [ ] **历史检验**：类似的观点在历史上是否被验证过？结果如何？

#### 信息完整性
- [ ] **反面证据**：文章是否呈现了可能推翻自己结论的证据？
- [ ] **负面后果**：文章是否讨论了方案的可能负面效果？
- [ ] **成本代价**：文章是否说明了实现建议方案需要什么代价？
- [ ] **适用范围**：文章的结论是否添加了适用条件和边界？还是放之四海而皆准？
- [ ] **时间维度**：文章是否考虑了长期后果？还是只看短期？

#### 可操作性
- [ ] **行动导向**：文章是否提供了可操作的行动指南？还是只停留在空谈？
- [ ] **知行统一**：文章是否把理论和实践结合起来？
- [ ] **条件说明**：如果建议行动，是否说明了所需条件和前提？

#### 时间ROI
- [ ] **独特信息**：这篇文章提供了什么你无法从其他地方轻易获得的信息？
- [ ] **决策相关性**：这篇文章是否与你当前的决策或学习目标直接相关？
- [ ] **认知增量**：阅读之后，你的认知是否会发生有意义的改变？

### 快速判断信号

| 低价值信号（D5 Fail） | 高价值信号（D5 Pass） |
|----------------------|----------------------|
| 观点无法被验证 | 可被实践检验 |
| 大量信息被省略 | 信息完整，主动呈现反面 |
| 只有理论，无可操作性 | 理论与实践结合 |
| 结论绝对化，无条件 | 结论有条件限制 |
| 与读者当前需求无关 | 提供独特且相关的洞察 |
