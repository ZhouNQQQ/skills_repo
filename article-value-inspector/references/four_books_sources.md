# 四书思想来源与理论基础

## 一、《批判性思维》核心贡献

**代表著作**：Brooke Noel Moore & Richard Parker《Critical Thinking》；Richard Paul & Linda Elder 批判性思维框架；Stephen Toulmin 论证模型

**核心贡献到 AVIM**：
1. **论证结构分析**：论证 = 结论 + 理由。没有理由支撑的结论只是断言。
2. **隐含假设识别**：任何论证都建立在未被明说的假设之上；假设崩溃则论证崩溃。
3. **谬误分类体系**：系统化的逻辑谬误识别框架，覆盖相关性谬误、弱归纳谬误、语义谬误等。
4. **图尔敏模型**：Data-Warrant-Claim-Backing-Rebuttal-Qualifier 六要素，检验真实世界论证的完整度。
5. **Paul-Elder 八要素模型**：Purpose-Question-Information-Concepts-Assumptions-POV-Inference-Implications，全景扫描思维质量。

**在 AVIM 中的位置**：主要支撑 **D3 逻辑维度** 和 **D2 事实维度** 的证据评估部分。

## 二、《学会提问》核心贡献

**代表著作**：M. Neil Browne & Stuart M. Keeley《Asking the Right Questions: A Guide to Critical Thinking》

**核心贡献到 AVIM**：
1. **海绵式 vs 淘金式思维**：从被动吸收到主动筛选的根本转变。
2. **十大批判性问题**：论题/结论/理由/歧义/假设/谬误/证据/替代原因/数据欺骗/省略信息——构成完整的论证审查链条。
3. **弱势 vs 强势批判性思维**：弱势是"证明别人错"，强势是"评估一切断言，包括自己的"。
4. **价值观假设与描述性假设**：区分作者未明说的价值偏好和事实信念。

**在 AVIM 中的位置**：贯穿全部五个维度，尤其是 **D3 逻辑维度**、**D4 假设维度** 和 **D5 实用维度**（省略信息检查）。

## 三、毛泽东思想核心贡献

**代表著作**：《实践论》《矛盾论》《改造我们的学习》《反对本本主义》《论持久战》

**核心贡献到 AVIM**：
1. **实事求是**："实事"是客观存在的一切事物，"是"是事物的内部联系，"求"是我们去研究。强调从客观实际出发，而非主观想象、情绪或教条。
2. **实践论**：实践是认识的来源、动力和检验标准。"实践、认识、再实践、再认识"的循环上升。观点必须可被实践检验。
3. **调查研究**："没有调查，没有发言权。" "一切结论产生于调查情况的末尾，而不是在它的先头。"
4. **矛盾论**：矛盾的普遍性和特殊性；主要矛盾和矛盾的主要方面；"两点论"与"重点论"的统一。
5. **透过现象看本质**："去粗取精、去伪存真、由此及彼、由表及里"的十六字诀——从感性认识上升到理性认识的具体操作方法。
6. **具体问题具体分析**："不同质的矛盾，只有用不同质的方法才能解决。" 反对教条主义和一刀切。

**在 AVIM 中的位置**：
- **D1 深度维度**："透过现象看本质"十六字诀是第一性原理的操作补充
- **D2 事实维度**："实事求是"和"调查研究"是事实审查的哲学基础
- **D5 实用维度**："实践论"是可检验性和可操作性的理论基础

## 四、《第一性原理》核心贡献

**代表来源**：亚里士多德《形而上学》起源；埃隆·马斯克等现代工程实践

**核心贡献到 AVIM**：
1. **原点思维**：将问题解构到最基本、不可再分的真理（物理定律、数学公理），然后从零重构。
2. **演绎优于类比**：类比推理只能产生微小迭代；第一性原理才能带来突破性洞察。
3. **假设质疑**：识别并挑战所有基本假设，区分"物理定律"和"人为约定"。
4. **苏格拉底式追问**：CRITIC 六步提问法和"五个为什么"法，追溯至不可再分的基本事实。
5. **可证伪性**：真正的知识必须是可以被证伪的；不可证伪的陈述不是知识。

**在 AVIM 中的位置**：主要支撑 **D1 深度维度** 和 **D4 假设维度**。

## 五、四书整合的逻辑

四本书看似来自不同传统（西方逻辑学、辩证法、古希腊哲学），但它们在 AVIM 中形成了互补的完整链条：

```
信息输入
    │
    ▼
【D1 深度】第一性原理 + 毛泽东思想「本质穿透」
    → 是否触及问题本源？还是只谈表象？
    │
    ▼
【D2 事实】毛泽东思想「实事求是/调查研究」+ 批判性思维「证据评估」
    → 是否有可靠的事实基础？
    │
    ▼
【D3 逻辑】批判性思维「论证分析」+ 《学会提问》「十大问题」
    → 论证是否逻辑严密？
    │
    ▼
【D4 假设】第一性原理「质疑假设」+ 《学会提问》「假设识别」
    → 隐含假设是否被挑战？
    │
    ▼
【D5 实用】毛泽东思想「实践论」+ 《学会提问》「省略信息」
    → 是否完整、可检验、可操作？
    │
    ▼
裁决输出（A/B/C/D + 行动建议）
```

这个链条的核心思想是：**先判断深度（是否触及本质），再检验事实（是否站得住脚），然后审查逻辑（是否推得出来），接着质疑假设（底层是否稳固），最后评估实用（是否完整可行动）。**

任何一个维度出现严重问题，都足以判定文章不值得投入时间精读。这确保了在信息过载时代的高效决策。
