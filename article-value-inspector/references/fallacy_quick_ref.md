# 逻辑谬误速查表

## 一、相关性谬误（前提与结论逻辑无关）

| 谬误 | 一句话识别法 | 文章中的信号词/模式 |
|------|------------|-------------------|
| **人身攻击 (Ad Hominem)** | 攻击人而非观点 | "他学历低/有前科/是外行，所以他说得不对" |
| **基因谬误 (Genetic)** | 因观点来源就否定/肯定 | "这是资本家的理论" / "这是官方说法" |
| **稻草人 (Straw Man)** | 歪曲对方再攻击 | 把对方观点极端化后再反驳 |
| **虚假两难 (False Dilemma)** | 只有两种选择 | "要么……要么……" / "不……就是……" |
| **滑坡谬误 (Slippery Slope)** | 一旦A就会滑到Z | "一旦允许A，接下来就会BCDEF……" |
| **诉诸公众 (Ad Populum)** | 大家都这么认为 | "90%的人都支持" / "全网都在说" |
| **诉诸情感 (Ad Misericordiam)** | 用情绪替代理由 | "不转不是中国人" / "想想孩子" |
| **诉诸权威 (Ad Verecundiam)** | 可疑权威跨界发言 | "某明星推荐" / "某专家说"（非专业领域） |
| **转移话题 (Red Herring)** | 答非所问，转移焦点 | "你总是挑我的错"（回应"你为什么撒谎"） |
| **光环效应 (Halo Effect)** | 因A好所以B也对 | "他是成功人士，所以他的观点都对" |

## 二、弱归纳谬误（证据不足却强行概括）

| 谬误 | 一句话识别法 | 文章中的信号词/模式 |
|------|------------|-------------------|
| **以偏概全 (Hasty Generalization)** | 个例推广全体 | "我身边的人都……" / "有个朋友……" |
| **事后归因 (Post Hoc)** | "A在B前，所以A导致B" | "自从实施了A，B就好了" |
| **因果混淆 (Confusing Correlation)** | 相关≠因果 | "喝咖啡的人长寿，所以喝咖啡长寿" |
| **忽略常见原因** | 忽略了共同原因C | "A和B同时增加，所以A导致B" |
| **过度简化因果** | 单一原因解释复杂事件 | "他失败是因为不够努力" |
| **类比谬误 (Weak Analogy)** | 类比基础不成立 | "国家财政就像家庭预算" |
| **一厢情愿 (Wishful Thinking)** | "我希望A，所以A会成真" | "相信就会成功" |

## 三、预设/语义谬误（语言陷阱）

| 谬误 | 一句话识别法 | 文章中的信号词/模式 |
|------|------------|-------------------|
| **偷换概念 (Equivocation)** | 同一个词换了意思 | 前半句的"自由"≠后半句的"自由" |
| **循环论证 (Begging the Question)** | 结论当理由 | "这本书可信，因为它内容真实" |
| **乱扣帽子 (Nominal Fallacy)** | 起名字≠解释 | "他有焦虑症"（没有解释原因） |
| **Complex Question** | 问题中暗藏假设 | "你什么时候停止打你老婆的？" |
| **歧义歧路 (Amphiboly)** | 句子结构导致歧义 | 语法上可多重解读的表述 |

## 四、削弱/逃避谬误（回避论证责任）

| 谬误 | 一句话识别法 | 文章中的信号词/模式 |
|------|------------|-------------------|
| **追求完美方案** | "不完美=不可用" | "这个政策还有漏洞，所以不该用" |
| **强求确定性** | "没有100%证据就不能做" | "不能完全排除风险，所以不行" |
| **诉诸无知 (Ad Ignorantiam)** | "不能证明假就是真" | "没有人能证明外星人不存在" |
| **两面讨好 (Double Standard)** | 对自己和对方用不同标准 | 同样的逻辑，正反都用来支持自己 |
| **回避问题 (Ignoratio Elenchi)** | 论证的不是真正的问题 | 看似在论证，实际上偏离核心议题 |

## 五、谬误信号词速查

看到以下词汇/句式，立即提高警惕：

- "大家都……" / "全网……" / "所有人都知道" → 诉诸公众
- "不……就是不爱国/不道德" → 诉诸情感
- "一旦……就会……就会……" → 滑坡谬误
- "要么……要么……" → 虚假两难
- "某专家说/某成功人士说" → 诉诸权威（检查领域相关性）
- "自从……之后……" → 事后归因
- "就像……一样" → 可能类比谬误
- "绝对" "必然" "毫无疑问" "100%" → 可能过度断言
- "这就是为什么……"（无实际解释） → 乱扣帽子
- "众所周知"（但没有引用） → 诉诸公众/缺乏证据

## 六、实战扫描口诀

```
人身稻草假两难，滑坡公众情感权威。
事后因果偏概全，循环偷换乱扣帽子。
追求完美求确定，两面讨好回避问题。
看到信号拉警报，逐一排查不慌张。
```
