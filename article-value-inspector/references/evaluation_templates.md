# 评估模板与评分体系

## 一、标准评估模板

当用户对一篇文章进行价值评估时，使用以下模板输出结果：

```markdown
# 文章价值检验报告

**文章**: [标题]  
**来源**: [网站/作者]  
**检验日期**: [YYYY-MM-DD]  
**检验框架**: AVIM 五维漏斗检查法

## 总体裁决

**等级**: [ A / B / C / D ]  
**建议**: [ 精读并笔记 / 选择性精读 / 略读即可 / 直接弃读 ]  
**时间预算**: [ X 分钟 ]

## 五维评分

### D1 本质/深度维度
- **评分**: [ Pass / Partial / Fail ]
- **证据**: [具体观察]
- **关键问题**: [触及本质？还是停留表面？]

### D2 事实/数据维度
- **评分**: [ Pass / Partial / Fail ]
- **证据**: [具体观察]
- **关键问题**: [来源可靠？数据真实？]

### D3 逻辑/论证维度
- **评分**: [ Pass / Partial / Fail ]
- **证据**: [具体观察]
- **关键问题**: [论证结构完整？有无谬误？]

### D4 假设/前提维度
- **评分**: [ Pass / Partial / Fail ]
- **证据**: [具体观察]
- **关键问题**: [假设被识别和挑战？]

### D5 实用/完整维度
- **评分**: [ Pass / Partial / Fail ]
- **证据**: [具体观察]
- **关键问题**: [可操作？信息完整？]

## 关键缺陷（导致降级的核心问题）
1. [问题描述]
2. [问题描述]

## 可取之处（如有）
1. [优点描述]

## 阅读策略建议
[具体建议]
```

## 二、快速评分矩阵

### 各维度权重

| 维度 | 权重 | 说明 |
|------|------|------|
| D1 深度 | 25% | 最核心的区分维度；触及本质 vs 停留表面 |
| D3 逻辑 | 25% | 论证有效性是基础门槛 |
| D2 事实 | 20% | 事实基础决定可信度 |
| D4 假设 | 15% | 假设挑战度反映思维深度 |
| D5 实用 | 15% | 完整性和可操作性 |

### 评分规则

| 评分 | 数值 | 说明 |
|------|------|------|
| Pass | 1.0 | 明显通过该维度检查 |
| Partial | 0.5 | 部分通过，有瑕疵 |
| Fail | 0.0 | 未通过该维度检查 |

### 等级换算

| 总分 | 等级 | 行动 |
|------|------|------|
| 0.85 - 1.00 | A | 精读并做笔记 |
| 0.65 - 0.84 | B | 选择性精读 |
| 0.40 - 0.64 | C | 略读即可 |
| 0.00 - 0.39 | D | 直接弃读 |

**硬性否决规则**（满足任一即降一级）：
- D1 (深度) = Fail 且 D3 (逻辑) = Fail → 自动 D 级
- 发现 3 种以上不同的逻辑谬误 → 最多 B 级
- 数据/来源存在明显造假或不可追溯 → 自动 D 级
- 纯粹情绪煽动，无任何事实支撑 → 自动 D 级

## 三、30秒极速判断法

当时间极其有限时，使用以下三问法进行快速判断：

1. **这是本质分析还是类比/表面描述？**（D1快速扫描）
   - 如果文章核心论证依赖类比或停留在现象描述 → 大概率低价值

2. **有理由支撑结论吗？来源可靠吗？**（D2+D3快速扫描）
   - 如果结论无理由支撑，或来源不明/有偏 → 大概率低价值

3. **作者是在证明一个观点，还是在煽动一种情绪？**（D3+D5快速扫描）
   - 如果情绪词汇远多于理性论证 → 大概率低价值

**30秒裁决**：
- 三问全部通过 → 进一步检查（B级以上潜力）
- 任意一问失败 → 直接弃读或仅略读（C级或D级）

## 四、不同文章类型的检查侧重

| 文章类型 | D1侧重 | D2侧重 | D3侧重 | D4侧重 | D5侧重 |
|----------|--------|--------|--------|--------|--------|
| **新闻报道** | 现象 vs 本质 | 来源可靠性 | 因果推断 | 媒体立场假设 | 信息完整性 |
| **学术论文** | 问题重要性 | 研究设计 | 论证严谨性 | 理论假设 | 局限说明 |
| **商业分析** | 根因分析 | 数据来源 | 逻辑链条 | 行业共识 | 可操作性 |
| **社论/评论** | 观点深度 | 事实引用 | 推理过程 | 价值观假设 | 反面意见 |
| **自媒体/短视频文案** | 本质触及 | 调查基础 | 谬误密度 | 共识挑战 | 煽动性 |
| **教程/指南** | 原理讲解 | 实践验证 | 步骤逻辑 | 前提条件 | 可操作性 |

## 五、个人阅读习惯建设建议

1. **建立信息食谱**：像管理饮食一样管理信息摄入。设定每日高质量阅读配额，拒绝低质填充。
2. **预设阅读目标**：打开任何文章前，先问自己"我需要从这篇文章获得什么？"
3. **淘金式批注**：阅读时在页边或笔记中记录质疑，而非只划重点。
4. **建立谬误敏感度**：每周刻意练习识别 3-5 篇文章中的谬误，培养直觉。
5. **追踪预测质量**：对于经常阅读的作者/媒体，追踪其历史预测的准确率，建立可信度档案。
6. **反向阅读训练**：主动寻找与自己观点相反的高质量文章，用强势批判性思维检验自己的信念。
